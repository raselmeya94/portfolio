# Project Cover Images — Replace Guide

Every project card on the portfolio shows a cover image from this folder.
Right now these are auto-generated placeholders (navy/teal gradient + initials)
so the layout looks complete. Whenever you have a real screenshot or logo,
just replace the file using the EXACT same filename below — no code changes
needed, the site will pick it up automatically.

Recommended image size: 1280x720 (16:9), JPG or PNG, < 500KB.
Screenshots of the actual UI work best. For PyPI/HF entries, a clean logo on
a flat background also works well.
